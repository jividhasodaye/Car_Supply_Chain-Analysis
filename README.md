# Car_Supply_Chain-Analysis
1️⃣ Project Title

Car Supply Chain & Logistics Analysis Dashboard

2️⃣ Project Description

This project presents an interactive Tableau dashboard that analyzes the car supply chain and logistics performance. The dashboard provides insights into key business metrics such as revenue, total units sold, shipping delays, discounts, customer feedback, and sales trends.

The goal of this project is to help understand supply chain efficiency, operational performance, and customer satisfaction in the automobile industry. By visualizing different aspects of the supply chain, the dashboard enables better decision-making and identification of operational bottlenecks.

This project demonstrates how data visualization tools can transform raw logistics data into actionable business insights.

🛠 Tech Stack

The following tools and technologies were used to build this project:

Tableau – Dashboard development and data visualization

SQL – Data querying and analysis

Microsoft Excel – Data cleaning and preprocessing

📂 Data Source

The dataset used for this project was obtained from:

Kaggle

The dataset contains information related to car sales, supply chain logistics, shipping modes, order quantities, customer feedback, and regional sales performance.

📊 Features & Highlights
📌 Business Problem

Automobile companies handle complex supply chain and logistics operations involving multiple car manufacturers, shipping methods, and regional sales markets. Tracking operational performance, delivery efficiency, and customer satisfaction can be difficult without proper analytical tools.

This dashboard helps businesses monitor sales performance, evaluate shipping efficiency, and understand customer feedback to improve logistics operations.

🎯 Objectives

Analyze overall revenue and units sold

Evaluate shipping efficiency and delivery delays

Identify top-performing car manufacturers

Analyze state-wise sales performance

Understand customer feedback distribution

Monitor sales trends over time

🔎 Insights

Certain car manufacturers generate significantly higher revenue than others.

Shipping mode impacts delivery efficiency, affecting customer satisfaction.

Bulk orders represent a large portion of total sales.

Some states contribute higher sales revenue, indicating stronger regional demand.

Customer feedback shows a mixed satisfaction level, highlighting opportunities for improvement.
